ELVISH is a bet that software can still be a public good: small, inspectable, and yours to run. We publish utilities on the open web, prioritize local execution, and refuse the usual extraction playbook. No growth team. No engagement OKRs. No vendor lock-in dressed as convenience.

If a flow touches your secrets, it should happen in your tab or on your disk — not in ours.

LOCAL FIRST

Core paths run client-side (WASM, JS, WebCrypto). We do not run your files through a black-box API for convenience. When something must touch the network, we say so — before you click.

NO RENT-SEEKING

AGPL keeps derivatives honest. The site is not a funnel to an “enterprise tier.” Accounts exist only where coordination truly needs identity — never as a tollbooth on math or encryption.

NO TELEMETRY BY DEFAULT

We ship with telemetry off unless an operator opts in. If enabled, it stays aggregate-only and anonymous: no user IDs, no message IDs, no IPs, no raw paths, and no behavioral profiling. Manual export only.

SIGNED, FORKABLE TRUTH

/log is changelog and receipt rail: minisign and OpenPGP where it matters. Verify builds. Mirror tarballs. Fork without asking permission.

QUIET CRAFT

Interfaces stay dense and legible — terminal cadence, monospace discipline, no dark patterns. We polish until it feels boring, because boring is what you trust with real work.

ANONYMOUS BY DEFAULT

The workshop stays unsigned as a brand on purpose: judge the artifacts, not a persona. Cryptographic identity shows up where cryptography belongs — keys, signatures, hashes — not in a hero headshot.
