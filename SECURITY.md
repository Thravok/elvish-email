# Security

Report vulnerabilities privately to the maintainers (see repository contact / security advisory channel on GitHub or GitLab).

Do not open public issues for exploitable findings before coordination.

**Scope:** `elvishapi`, `elvishmta`, `elvishworker`, browser apps (`apps/web`, `apps/admin`), and documented deployment configs.

**Out of scope:** third-party dependencies without a demonstrable impact on ELVish defaults; issues in forks or unmaintained `ui/` trees removed from this repo.
